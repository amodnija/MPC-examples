This folder contains supplemental materials for the book:

Fundamentals of Linear Control
A Concise Approach
Mauricio de Oliveira
http://www.cambridge.org/deOliveira

and are released under an MIT license (see license.txt)

The m files will allow you to generate all figures and perform many of
the calculations that you find in the book using MATLAB.

If you have MATLAB R2106b or later you might want also to download
Live Script versions of these files from the book website.

Mauricio de Oliveira
